#  Research Question

## Primary Research Question
How effectively can a YOLO11-based multi-class object detection pipeline identify façade defects from inspection imagery to support scalable condition assessment workflows?

## Secondary Questions
1. Which façade defect classes are hardest to detect?
2. How does dataset balance affect class-wise AP?
3. Can detections support future BIM-linked maintenance workflows?
4. What limits transferability to unseen façades?

## Evaluation Lens
- mAP@0.5
- class-wise AP
- recall on weak classes
- real unseen façade inference quality
